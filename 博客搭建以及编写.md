# 在本地写文章

* 1:使用 vscode 打开 xinzhuzi.github.io
* 2:终端打开, 
```
cd xinzhuzi.github.io
```
* 3:开始写文章,
```
hexo new 文章名字
```
* 4:写完之后输入
```
hexo clean
hexo g
hexo d
```
* 5:使用 Github Desktop 上传到 GitHub 网站
