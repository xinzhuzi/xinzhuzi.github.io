## 天道无常，劫数难逃

自上古时期，天地灵气充裕，修仙之风盛行。然天道无常，劫数难逃。每隔千年，天地便有一次大劫降临，修仙者称之为「千年劫」。

### 世界观

**千年志·道劫**的世界中，修仙分为三大境界：

| 境界 | 描述 |
|------|------|
| **炼气期** | 引气入体，淬炼经脉，凡人踏入修仙之路的第一步 |
| **筑基期** | 凝练道基，铸就金丹，初窥仙道门径 |
| **金丹期** | 金丹大成，法力通天，可御空而行 |
| **元婴期** | 破丹成婴，元神出窍，成就一方仙师 |
| **化神期** | 元神化形，开宗立派，威震一方 |
| **渡劫期** | 经历天劫，渡过则飞升成仙，不过则灰飞烟灭 |

### 游戏剧情

玩家将扮演一名在乱世中求存的凡人，因缘际会踏入修仙之路。历经宗门纷争、妖魔之乱、天劫降临，最终能否突破桎梏，成就仙道？

一切，皆在你手中。
